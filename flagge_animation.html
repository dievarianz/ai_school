Schweizer Flagge - Animationsvorschläge

1. Wehende Flagge

/* CSS */ @keyframes wave { 0% { transform: rotate(0deg); } 25% { transform: rotate(5deg); } 75% { transform: rotate(-5deg); } 100% { transform: rotate(0deg); } } .flag-waving { animation: wave 2s ease-in-out infinite; transform-origin: center left; }

2. Pulsierende Flagge

/* CSS */ @keyframes pulse { 0% { transform: scale(1); } 50% { transform: scale(1.1); } 100% { transform: scale(1); } } .flag-pulsing { animation: pulse 2s ease-in-out infinite; }

3. Rotierende Flagge

/* CSS */ @keyframes rotate { 0% { transform: rotateY(0deg); } 100% { transform: rotateY(360deg); } } .flag-rotating { animation: rotate 3s linear infinite; transform-style: preserve-3d; }

4. Leuchtende Flagge

/* CSS */ @keyframes glow { 0% { box-shadow: 0 0 5px rgba(255, 0, 0, 0.3); } 50% { box-shadow: 0 0 20px rgba(255, 0, 0, 0.7); } 100% { box-shadow: 0 0 5px rgba(255, 0, 0, 0.3); } } .flag-glowing { animation: glow 2s ease-in-out infinite; }

5. 3D-Flip Animation

/* CSS */ @keyframes flip { 0% { transform: perspective(400px) rotateY(0); } 100% { transform: perspective(400px) rotateY(180deg); } } .flag-flipping { animation: flip 3s linear infinite; transform-style: preserve-3d; }

6. Schwebende Flagge

/* CSS */ @keyframes float { 0% { transform: translateY(0); } 50% { transform: translateY(-10px); } 100% { transform: translateY(0); } } .flag-floating { animation: float 3s ease-in-out infinite; }
